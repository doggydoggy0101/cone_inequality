### Conjecture

adjusted inequality: $H(x,y)\leq_{\mathcal{L}_\theta}TA(x,y)$, 
where $\theta=\dfrac{\pi}{3}$ and $T=\begin{pmatrix}\tan(\theta)&\mathbf{0}\\\mathbf{0}^T&\mathbf{1}\end{pmatrix}$
